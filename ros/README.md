# ros_music_adapter

## Software Dependencies
- MUSIC (https://github.com/INCF/MUSIC)
- ROS (http://wiki.ros.org/ROS/Installation)
- CMake
- OpenMPI
- GSL BLAS
- Json-cpp
- pthread

##### Additional Software
- Gazebo (http://gazebosim.org/)
- NEST (http://nest-simulator.org/)
- NEURON (http://neuron.yale.edu/neuron/)

- cmake .
